    Sobre Mim

    Meu nome é Vitor Levi, tenho 17 anos e estou no terceiro ano do Ensino Médio em uma escola técnica, cursando Desenvolvimento de Sistemas. Programo há dois anos e atuo como desenvolvedor full stack, trabalhando com front-end e back-end. Busco entregar soluções com foco em performance, usabilidade e qualidade, entendendo o sistema de forma completa, desde a modelagem de dados até a experiência do usuário. Sou dedicado, curioso e apaixonado por aprendizado contínuo, sempre explorando novas tecnologias para tornar meus projetos mais eficientes e escaláveis.

    Tecnologias

    HTML5

    Linguagem de marcação usada para estruturar o conteúdo da web, como textos, imagens, links e formulários. É a base de toda página web.

    CSS3

    Folhas de estilo responsáveis por definir o visual das páginas HTML, como cores, espaçamentos, fontes e posicionamento dos elementos.

    JAVASCRIPT

    Linguagem de programação que permite adicionar interatividade às páginas web, como animações, validações de formulários e manipulação do DOM.

    REACTJS

    Biblioteca JavaScript focada na construção de interfaces de usuário. Utiliza componentes reutilizáveis e oferece alta performance com renderização eficiente.

    STYLED COMPONENTS

    Biblioteca de estilização para React que permite escrever CSS diretamente dentro dos componentes JavaScript, promovendo encapsulamento e modularidade.

    NODEJS

    Ambiente de execução JavaScript no servidor. Permite construir aplicações backend escaláveis usando a mesma linguagem do frontend.

    EXPRESSJS

    Framework web minimalista para Node.js, usado para construir APIs e servidores de forma simples, rápida e flexível. É leve, mas poderoso, e permite organizar rotas, middlewares e integrações com facilidade.

    MYSQL

    Sistema de gerenciamento de banco de dados relacional. Utiliza linguagem SQL para armazenar e manipular dados estruturados com alta confiabilidade.

    MONGODB

    Banco de dados NoSQL orientado a documentos. Flexível, escalável e ideal para aplicações modernas que lidam com dados dinâmicos e não estruturados.

    GIT

    Sistema de controle de versão distribuído. Permite gerenciar o histórico de alterações no código, colaborar em equipe e reverter mudanças quando necessário.

    POSTMAN

    Ferramenta para testar e documentar APIs REST. Facilita o envio de requisições HTTP, visualização de respostas e automação de testes.
